# PaperBanana Clients

PaperBanana multi-client monorepo for shared client code and platform apps.

## Repository Structure

- `packages/api/` - shared backend API client wrappers.
- `packages/types/` - shared TypeScript type definitions.
- `packages/business/` - shared pure business logic without UI dependencies.
- `packages/ui-core/` - shared React UI components.
- `packages/design-tokens/` - shared colors, typography, spacing, and design variables.
- `apps/web/` - web client placeholder.
- `apps/desktop/` - Electron desktop client placeholder.
- `apps/android/` - Capacitor Android client placeholder.

Mini program code will be handled separately.
